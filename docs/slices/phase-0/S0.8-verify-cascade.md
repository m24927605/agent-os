# SLICE-P0-008: verify 成為 polyglot 級聯入口（fail-closed，非偽綠）

- **Phase**: P0 周邊（roadmap §2「待新增 slice」；為 P1 Go / P3 Python 進場預先把 gate 綁住）
- **Branch**: slice/p0-008-verify-cascade
- **Author**: <id>    **Adversarial reviewer**: <須為 fresh-context、非作者>
- **Size budget**: 估計 <= 0.5 day；net LOC <~90、files <~4（2 scripts + package.json + 1 test）

## (1) ID + Title
SLICE-P0-008 — 讓 `pnpm run verify` 級聯呼叫 `verify:go` / `verify:py`，使 P1 Go kernel / P3 Python SDK 一進場就**無法繞過**統一 gate；佔位必須 **fail-closed**（語言根不存在→skip exit 0；存在但 gate 未設定→fail exit≠0），絕不偽綠。

## (2) Goal（一句話）
新增 `scripts/verify-go.sh`、`scripts/verify-py.sh`（fail-closed cascade）+ 把 `verify:go`/`verify:py` 串進 `verify`，並以 contract 測試鎖定「已串接」「存在卻未設定 → fail」。

## (3) In-scope / Out-of-scope
- In-scope:
  - `scripts/verify-go.sh` / `scripts/verify-py.sh`：plane 目錄（預設 `kernel/`、`python/`，可由 `VERIFY_GO_PLANE`/`VERIFY_PY_PLANE` 覆寫以利測試）不存在 → `skip exit 0`；存在但缺 gate 設定（go.mod/.golangci 或 pyproject/ruff 設定，或 toolchain 未裝）→ **fail exit≠0**；齊全 → 跑該語言 gate。
  - `package.json`：新增 `verify:go`/`verify:py`，串進 `verify`（在 `deps:check` 之後、`secret-scan` 之前）。
  - contract 測試 `src/build/verify-cascade.test.ts`。
- Out-of-scope:
  - 真實 Go/Python gate 內容（depguard/staticcheck/import-linter 細節）→ 由 **P1 kernel slice / P3 SDK slice** 補上（屆時 plane 目錄出現，cascade 自動要求其 gate）。

## (4) Design delta + modules + public interface + dependency direction
- **Design delta**: `verify` 從 TS-only 變成 polyglot 級聯入口；新增的子關卡 fail-closed。
- **Modules touched**:
  - `scripts/verify-go.sh` / `verify-py.sh` — **唯一責任**：對各語言 plane 做 fail-closed 偵測與 gate 級聯。
  - `package.json`（scripts）— 宣告級聯組成。
- **PUBLIC interface（命令面）**: `pnpm run verify:go` / `pnpm run verify:py`（exit 0 = skip 或 gate 通過；exit≠0 = 存在卻未設定 / gate 失敗）。
- **Dependency direction**: 純 build/CI 腳本，無 runtime 模組依賴；`deps:check` 不受影響。

## (5) Test-first plan（先寫的 RED 測試）
- 測試檔: `src/build/verify-cascade.test.ts`
- RED（infra/governance slice：RED = 失敗的 contract 斷言）:
  - [ ] `package.json` 的 `verify` 串接了 `verify:go` 與 `verify:py`（未串接前 → 紅）。
  - [ ] plane 目錄不存在 → 腳本 exit 0（skip）。
  - [ ] plane 目錄存在但未設定 gate → 腳本 exit≠0（fail-closed，**禁偽綠**）。
- 首次紅燈（腳本/串接尚未存在）:

  ```
  ... FAIL  expected verify script to contain "verify:go"

  ```
